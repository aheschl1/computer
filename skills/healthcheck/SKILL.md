# Health Check

## When to use this skill
Use this skill when the user needs to perform a baseline health check of the system.

## How to use this skill
1. Invoke the skill by running the health_check.sh file, and reviewing the output for any issues or anomalies.
2. If any issues are detected, investigate further to identify the root cause and take appropriate action to resolve the issue.
