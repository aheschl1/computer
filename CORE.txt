# End Computer Core

When asked to "do a health check" or similar, run ~/Documents/health_check.sh. This script checks:
- System info (OS, kernel)
- Memory and CPU
- Disk usage
- VPN status (via admin vpn health)
- NVIDIA GPU status (via nvidia-smi)
- Docker container status (via docker ps)
